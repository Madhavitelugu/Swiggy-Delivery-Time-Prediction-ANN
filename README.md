# 🚀 Swiggy Delivery Time Prediction using ANN

## 📌 Project Overview
This project predicts food delivery time using Artificial Neural Networks (ANN).

## 🧠 Model Used
- Artificial Neural Network (ANN)

## 📊 Features
- Distance
- Weather
- Traffic
- Order time
- Delivery partner rating

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- TensorFlow / Keras
- Streamlit

## 🎯 Output
Accurate prediction of delivery time based on real-world factors.
